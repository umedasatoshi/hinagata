introducing HINAGATA.

HINAGATAはWEB制作を始めるためのシンプルでオーソドックスなファイル群です。

コーダーやフロントエンジニアにとって、どんなテンプレートがあればコーディングを始めやすいだろう？
その答えの一つが、機能を削ぎ落としていきシンプルで、かつオーソドックスな構成のファイル群でした。

HINAGATA projects.

HINAGATA for Wordpress

HINAGATA for Wordpressは使用頻度の高いファイルだけで構成されたテンプレートです。
便利な固有機能はなく、また複雑なコードもありません。構築しやすいスタンダードなテンプレートです。
